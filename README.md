# 🌦 Weather Prediction using Machine Learning

## 📌 Overview
Machine Learning project to predict temperature using Random Forest and analyze seasonal trends

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 📊 Workflow
1. Data Cleaning & Preprocessing  
2. Feature Engineering (month, day extraction)  
3. Model Training  
4. Evaluation (MAE)  
5. Visualization  

## 🤖 Models Used
- Linear Regression  
- Random Forest Regressor  

## 📈 Key Insights
- Temperature shows strong seasonal trends  
- Peak temperatures occur mid-year (summer months)  
- Random Forest improved prediction accuracy compared to Linear Regression
